# Sistema de Chamado de Veículos para Hotel

Este é um sistema desenvolvido para gerenciar o chamado de veículos para um hotel. A aplicação permite que a equipe do hotel registre as solicitações de transporte feitas pelos hóspedes, organizando e facilitando o controle de veículos e motoristas.

## Funcionalidades

- Registro de solicitações de veículos.
- Controle de veículos disponíveis e motoristas.
- Interface simples e fácil de usar para a equipe do hotel.
- Notificações e atualizações de status de chamados.

## Instalação

1. Faça o download do projeto [aqui](https://github.com/HudsonDomin/unique-scv/releases/tag/1.0).
2. Extraia os arquivos em um diretório de sua escolha.
3. Siga as instruções de configuração abaixo.

## Como Usar

 - Abra o executavel "unique-scv.exe".
Uma janela cmd abrirá, esse é o servidor.

## Para outros computadores se conectarem
 - Descubra o ipv4 do computador que está executando o servidor, exemplo: 10.0.0.112
 - Em outro computador, na barra de endereço do navegador, isira o ip do computador que está executando o servidor + :8000
 - Exemplo: 10.0.0.112:8000
